# ScriptableUser
Scriptable User Engine Remake (SUEngine).

A brand-new version, made for newer computers.

The last one didn't fully pushed though.

A Scriptable Virtual User which allows you to perform some actions automaticlly so do more easily while you're away.

For example, Genshen Impact Autoplay, block pop-up windows, Or just not letting you be kicked out from a server.

Running on windows since it's developed using some WINAPI. The GUI part is win32 style, while some of them will be made under Qt.
